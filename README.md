
# IDC EVO and MINI IPA Test Automation

IPA Test Automation tool to run automated voice and speech test and peform KPI's for IDV evo platforms and MINI platforms


## Tech Stack
- python
- pandas
- ffmpg
- pyttsx3
- gTTS
